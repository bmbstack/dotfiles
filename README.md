# Dotfiles #

My dotfiles for:

* vim
* tmux
* zsh(with theme)
* git
* sublime

## Quick Install & Uninstall ##

Before Install, make sure you have installed zsh.

###注：
- python (2.7+ or 3.0+)
- Bash-4.0+ or Zshell.
- neocomplete requires Vim 7.3.885 or later with Lua support ("+lua")，with Python support ("+python")

Install:

	./install.sh
    source ~/.zshrc
    zplug load

Uninstall:

	./uninstall.sh
